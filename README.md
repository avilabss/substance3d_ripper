# Substance 3D Ripper

```bash
usage: ripper.py [-h] --ims_sid IMS_SID --collection_id COLLECTION_ID

Substance 3D Ripper CLI

options:
  -h, --help            show this help message and exit
  --ims_sid IMS_SID     IMS session ID for authentication. Find it in your browser by entering developer tools. Navigate to Application > Cookies >
                        https://substance3d.adobe.com/ and copy the value of ims_sid.
  --collection_id COLLECTION_ID
                        ID of the collection to retrieve
```
